| 类型     | 含义示例                             |
| -------- | ------------------------------------ |
| feat     | 新增功能（feature）                  |
| fix      | 修复 bug                             |
| refactor | 重构（既不是新增功能也不是修 bug）   |
| docs     | 文档更新                             |
| style    | 格式改动，不影响功能（如空格、缩进） |
| chore    | 杂项改动（更新依赖、配置等）         |
| test     | 添加或更新测试                       |

```
eg.
feat: 初步完成关卡UI框架

- 增加关卡选择面板
- 接入按钮事件（暂未接入数据）

git config --global core.editor '"D:\Programming\IDE\Microsoft VS Code\Code.exe"'
```
